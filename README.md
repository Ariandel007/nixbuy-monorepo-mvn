# English

This is a project with the only intention to do a mono repo of microservices which each one uses different architectures approaches.

## Tech
The following is being used:
- Maven
- Spring Boot
- Spring JPA
- Resilence4j
- Zipkin
- Debezium
- PostgreSQL
- Spring Cloud Gateway
- Apache Kafka
- Rest Assured
- Junit
- TestContainers
